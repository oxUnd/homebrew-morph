# OxUnd Morph

## How do I install these formulae?

`brew install oxund/morph/<formula>`

Or `brew tap oxund/morph` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "oxund/morph"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
